void init_ring_buffer ( ring_buffer_t * buffer,
uint32_t buf_size )

Initializes the ring buffer pointed to by buffer.

This function can also be used to empty/reset the buffer. The resulting buffer can contain buf_size-1 bytes.

Parameters
bufferThe ring buffer to initialize.
buf_sizeThe size of the allocated ringbuffer.
